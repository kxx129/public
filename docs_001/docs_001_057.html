57　ＳＷＯＴ分析

会話１ 2024年9月1日 14:00
Ａ  今日は、今後の経営戦略を立てるために、当店のＳＷＯＴ分析を行いたいと思います．
ＳＷＯＴ分析とは、経営環境を判断するために内部的な「強み（Strength）」　　　　　「弱み（Weakness）」と外部的な「機会（Opportunity）」「脅威（Threat）」を分析し、その結果を検討したうえでこれから自社が進んでいく方向を定めようとする分析　手法です。まずは内部環境の「強み」「弱み」を挙げてください。
Ｂ  当店は地元密着型のスーパーですので、「強み」としては、知り合いの農家から産地直送の野菜を仕入れることができる、昔からのなじみ客が多いといったところ　　　でしょうか。「弱み」は、大きな駐車場がないこと、品揃えが大型スーパーには　　　　かなわないことです。
Ａ  それに、OA化の遅れにより事務効率が低いことも「弱み」として挙げられますね。外部環境の「機会」「脅威」についてはどうですか。
Ｂ  「機会」は、近くに大型マンションが建つこと、「脅威」は、駅前にコンビニができたこと、チェーン展開している格安スーパーの進出計画があることでしょう。

Ａ  実は今、キャリアアップのために転職を考えています。成功する秘訣はありますか。
Ｂ  自分自身のＳＷＯＴ分析をするといいですよ。ＳＷＯＴ分析は、企業が経営戦略を立てるときに使われるものですが、個人を分析する手法としても使えます。
Ａ  自分の「強み」や「弱み」を整理するのですね。
Ｂ  はい。自分の能力はどの方面で強く、どの方面で弱いかについて、項目ごとに五段階で自己診断します。その結果をレーダーチャートにすると、自分の能力が一目瞭然ですよ。次は、自分を取り巻く外部の環境について考えます。現在所属する組織、経済状況、規制、予測されるビジネスチャンスなどです。それらの要素を自分にとって「機会」であるのか「脅威」であるのか振り分けます。こうすることで、今、自分自身の置かれている状況が分り、今後、どう進むべきなのかが明確になりますよ。
